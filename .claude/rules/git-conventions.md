# Git Conventions

## Commit Messages

- Format: `type: description`
- Types:
  - `data` — adding or updating career data (jobs, skills, achievements)
  - `template` — changes to resume templates
  - `resume` — generated resume output
  - `config` — changes to project config or Claude rules
  - `docs` — README or documentation updates

## Examples

- `data: add senior engineer role at Acme Corp 2022-2024`
- `data: update skills inventory with cloud certifications`
- `template: add hybrid resume format`
- `config: update resume writing rules`
