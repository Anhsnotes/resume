# Llama 3 Self-Hosted API

- Repo: github.com/Anhsnotes/llama3_api (private)
- Languages: Python, Dockerfile
- Status: Completed

## Description

A self-hosted Llama 3 API server running in Docker with GPU support. Provides a REST endpoint for text generation, deployable on both Linux and Windows with NVIDIA GPU passthrough.

## Tech Stack

- Llama 3 (Meta LLM)
- Docker Compose with GPU passthrough (NVIDIA CUDA)
- Python (FastAPI or similar REST server)

## Key Capabilities Demonstrated

- Self-hosted LLM deployment
- GPU-accelerated inference via Docker
- REST API design for AI model serving
