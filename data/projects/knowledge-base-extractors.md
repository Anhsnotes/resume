# Knowledge Base Extractors

- Repo: github.com/Anhsnotes/knowledge-base (private)
- Languages: Jupyter Notebook, TSQL, Python
- Status: Completed

## Description

Tools to extract knowledge from business documents, dashboards, data models, and processes. Includes specialized extractors for SQL, JSON, XML (Tableau workbooks), flowcharts (draw.io), and data mining.

## Extractors

- SQL Extractor — crawls and parses SQL scripts
- JSON Extractor — parses JSON structures into tabular/relational format (Python + SQL Server OPENJSON)
- XML Extractor — extracts metadata from Tableau workbooks (.twbx)
- Flowchart Extractor — parses draw.io diagrams into structured data
- Data Mining module

## Key Capabilities Demonstrated

- Metadata harvesting from diverse source formats
- Tableau workbook reverse engineering
- Business process documentation automation
- Enterprise data catalog tooling
