# Knowledge Base with Ontology

## Description

An upgraded version of the data_warehouse project that adds Ontology for controlling the meanings of data. Bridges the gap between raw data and business vocabulary.

## Tech Stack

- Python
- Docker / Shell scripts
- Ontology framework for semantic data control

## Key Capabilities Demonstrated

- Ontology-driven data governance
- Semantic layer design for enterprise data
- Evolution from data warehouse to knowledge-centric architecture
