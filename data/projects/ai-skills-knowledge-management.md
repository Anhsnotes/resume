# AI Skills / Knowledge Management System

- Repo: github.com/Anhsnotes/ai_skills (public)
- Languages: Python
- Status: Active

## Description

A knowledge management system with an AI engine for ingesting, chunking, evaluating, and amending knowledge from business documents. Includes a CLI, API, batch processing, and a decision/evaluation engine. Also contains a Claude Code SQL compiler skill.

## Tech Stack

- Python (CLI, API, engine)
- SQLite or Postgres (knowledge storage)
- AI ingestion pipeline (extractor → chunker → pipeline)
- Decision/evaluation engine (evaluator, amendment, decision modules)
- Claude Code skills (SQL compiler with dbt integration)

## Key Capabilities Demonstrated

- Knowledge extraction and management systems
- AI-powered ingestion pipelines (extract → chunk → store)
- Decision engine architecture (evaluate → decide → amend)
- Claude Code skill authoring (SQL compiler for dbt projects)
- CLI and API design for developer tools
