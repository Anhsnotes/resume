# Echolify

- Repo: github.com/Anhsnotes/echolify (public) + private variants (echolify-llm-rag, echolify-rag, echolify-prod)
- Languages: JavaScript, HTML
- URL: www.echolify.com
- Status: Active

## Description

A RAG-powered AI application built on Cloudflare Workers AI. Uses Cloudflare D1 (database), Vectorize (vector embeddings), and Workers AI (LLM inference) to provide retrieval-augmented generation. Multiple iterations from basic RAG to production deployment.

## Tech Stack

- Cloudflare Workers / Workers AI
- Cloudflare D1 (serverless SQL)
- Cloudflare Vectorize (vector search)
- Hono (web framework for Workers)
- LangChain (text splitting)
- Anthropic Claude / Llama models

## Key Capabilities Demonstrated

- RAG architecture (retrieval-augmented generation)
- Serverless AI application deployment
- Vector database and embedding pipelines
- Edge computing with Cloudflare Workers
- Iterative product development (v1 → RAG → production)
